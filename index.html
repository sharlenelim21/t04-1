SVG Shape Coordinates: Annotated Examples

How SVG Coordinates Work

  • <rect>: x and y set the top-left corner. width and height set size.
  • <circle>: cx and cy set the center. r sets the radius.
  • <ellipse>: cx, cy set the center. rx, ry set radii.
  • <line>: x1, y1 set the start; x2, y2 set the end.
  • <polyline>: points is a list of x,y pairs for each vertex.
  • <polygon>: points is a closed shape from x,y pairs.
  • <path>: d attribute uses commands (M, Q, T, etc.) for complex shapes.
  • <text>: x, y set the text anchor point.

Each annotation above shows the coordinates used in the code and where they appear in the drawing. This helps you match SVG code to visual output.
